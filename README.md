🎙️ AudioKit


AudioKit est une application web qui génère des audio-guides touristiques immersifs et personnalisés, propulsée par l'IA Gemini 2.5 Flash de Google.

✨ Fonctionnalités

🗺️ Génération d'un script narratif adapté au lieu, au public et au style choisi
🎭 4 styles de guide : Conférencier, Vieux Sage, Indiana Jones, Local
👨‍👩‍👧 3 publics cibles : Enfants, Ados, Adultes
⏱️ Durée ajustable de 5 à 30 minutes
🎵 Mixage optionnel avec une ambiance sonore
📄 Enrichissement du script via un document PDF source
📍 Coordonnées GPS intégrées dans les métadonnées du fichier MP3
📥 Téléchargement direct du guide en MP3


🛠️ Stack technique

Streamlit — Interface web
Google Gemini 2.5 Flash — Génération du script
Edge TTS — Synthèse vocale
pydub + FFmpeg — Mixage audio


🚀 Utilisation

Connectez-vous avec le mot de passe
Configurez vos paramètres dans la sidebar
Saisissez le nom du monument ou du lieu
✍️ Rédigez le script
📝 Révisez et ajustez si besoin
🔊 Générez l'audio final
📥 Téléchargez votre guide !
